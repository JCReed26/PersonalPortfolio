Best Play Option Predictior

Next.js, Python, PostgreSQL, Docker

Team: Jimmy, Marat

This project I worked with a friend Marat to create a football real time play prediction game simulator, where users choose their team and input changes to a live game. I developed a secure and scalable football game simulation platform by building RESTful API with Python and FastAPI to handle client-server interactions. For the prediction model I implemented a simple Pytorch Forward Neural Network trained with open source play data from the past 3 NFL seasons. Receipt Reader and Categorization API

Python, Tesseract-OCR, OpenAI, Docker

This program takes in an image opens, manipulates, and reads the text from the image with CV2 and PIL libraries and reads the text using Tesseract-OCR with the pytesseract library. The program then categorizes the items on the receipt by sending the data read into a Chat GPT 4o prompt to get rid of all of the extra text, categorize the purchase, get the items, prices, and total cost. This program is the first step in a larger project to automate budgeting and expense tracking.
